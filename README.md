# 🎬 Netflix Movies Analysis

> **Exploratory Data Analysis on 9,827 Netflix movies** — uncovering genre trends, audience behavior, popularity patterns, and language diversity across a century of cinema (1902–2024).

## 📌 Project Overview

Netflix is the world's leading streaming platform with **260+ million subscribers globally**. But what does its movie catalog actually look like?

This project dives deep into Netflix's film library using data from **The Movie Database (TMDB)** to answer questions about what genres dominate, which films resonate most with audiences, how content has evolved over time, and how linguistically diverse the platform really is.

Whether you're a data enthusiast, a Netflix binge-watcher, or a recruiter — this project has something for you. 🍿

## 📂 Dataset Summary

| Feature | Details |
|---------|---------|
| **Source** | The Movie Database (TMDB) |
| **Total Records** | 9,827 movies |
| **Time Period** | 1902 – 2024 |
| **Languages** | 43 unique languages (77% English) |
| **Genres** | 19 unique genres |

## 📊 Key Findings

- 🎭 **Drama** is the most frequent genre on Netflix by a significant margin
- 🔥 **Spider-Man: No Way Home** holds the highest popularity score (5083.95)
- 🗳️ **Action & Adventure** genres receive the most audience votes
- 📅 **2020** saw the peak in Netflix movie releases
- 🌍 Despite 43 languages, **English dominates at 77%** of the catalog
- 📈 **Vote count is a stronger predictor of popularity** than vote average alone
