# 🎬 Netflix Movies Analysis

> Exploratory Data Analysis on 9,827 Netflix movies from TMDB (1902–2024)

---

## 📌 Overview
This project analyzes Netflix's movie catalog to uncover trends in genres,
popularity, audience ratings, and language diversity.

## ❓ Key Questions Answered
- 🎭 What is the most frequent genre on Netflix?
- 🗳️ Which genre receives the most audience votes?
- 🔥 Which movie has the highest popularity?
- 📅 Which year had the most movie releases?
- 🌍 How diverse is Netflix's language catalog?

## 🔧 Tools & Libraries
| Tool | Purpose |
|------|---------|
| Python | Core language |
| Pandas | Data cleaning & analysis |
| Matplotlib | Visualizations |
| Seaborn | Styled charts |
| NumPy | Numerical operations |

## 📊 Dataset
- **Source:** The Movie Database (TMDB)
- **Records:** 9,827 movies
- **Period:** 1902 – 2024
- **Languages:** 43 unique languages
- **Genres:** 19 unique genres
